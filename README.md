# Windows-Media-Player-9-WebApp-Clone
A nostalgic web recreation of Windows Media Player 9 Series. Features local audio/video playback, YouTube &amp; HLS radio streams, 10-band EQ, spatial audio, beat-synced visualizations, in-browser MKV remuxing via FFmpeg.wasm, voice control, recording, and 12+ custom skins. happily but occasionally angrily built with vanilla JS, HTML, and CSS!.
